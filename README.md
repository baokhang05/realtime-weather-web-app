# Ứng dụng Web tra cứu thời tiết thời gian thực

**Vai trò:** Lập trình Logic & Tối ưu UI/UX

## Tổng quan dự án
Xây dựng một ứng dụng web cung cấp thông tin thời tiết theo thời gian thực. Dự án tập trung vào việc xử lý dữ liệu API, tối ưu hóa giao diện người dùng trên nhiều thiết bị và quản lý dữ liệu cá nhân hóa qua Cloud Database.

## Chi tiết kỹ thuật & Tính năng
- **Tích hợp API:** Khai thác và xử lý dữ liệu từ API bản đồ thời tiết (WindyMap) để hiển thị thông tin chính xác.
- **Backend & Cơ sở dữ liệu:** Thiết lập cấu hình dịch vụ trên nền tảng Firebase. Xây dựng luồng đăng nhập xác thực và đồng bộ hóa dữ liệu lịch sử tìm kiếm của người dùng thông qua Cloud Firestore.
- **Giao diện (Frontend):** Lập trình các tính năng xử lý dữ liệu cốt lõi (đọc nhiệt độ, tốc độ gió) và tối ưu hóa hiển thị (Responsive Design) đảm bảo trải nghiệm mượt mà trên cả thiết bị di động và máy tính.
